Explicacion del proyecto

Se busca aplicar los conocimientos vistos durante las clases, el fin de este reto es conectarnos a una API y mostrar lo siguiente:

  • Presente un muestra de la información de la API
  • Generar una funcionalidad de búsqueda
  • Tener un botón el cuál permita limpiar la búsqueda
  • Mostrar cuantas búsquedas ha realizado el usuario durante la ejecución
  • Mostrar el histórico de las búsquedas realizadas

Extra:

  • Poder visualizar el historico por segunda vez
  • Las búsquedas deben ser con datos locales
